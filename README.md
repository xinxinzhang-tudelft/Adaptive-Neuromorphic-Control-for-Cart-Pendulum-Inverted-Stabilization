# Adaptive Neuromorphic Control for Inverted Stabilization of a Cart–Pendulum System

This repository provides MATLAB implementations and simulation results for the paper:

**“Adaptive Neuromorphic Control for Inverted Stabilization of a Cart–Pendulum System”**
(IFAC Congress 2026)

---

## 📂 Repository Structure

The project is organized into three main simulation scripts, each corresponding to a specific experimental case:

* `case1.m` — Simulation for Case 1
* `case2.m` — Simulation for Case 2
* `case3.m` — Simulation for Case 3

Each script runs independently and reproduces the results presented in the paper.

## ⚙️ Requirements

* MATLAB (R2020 or later recommended)
* No additional toolboxes required unless specified in individual scripts

---

## 📊 Description

Each simulation demonstrates the performance of the proposed **neuromorphic controller** applied to an inverted cart–pendulum system. The scripts include:

* System dynamics modeling
* Dual Half-Center Oscillator (HCO) control implementation
* Energy-based adaptation mechanism
* Visualization of system states and control signals
